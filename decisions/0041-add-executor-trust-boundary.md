# ADR-0041: Add Executor Trust Boundary

**Date**: 2026-04-27
**Status**: Accepted
**Pipeline**: pattern-lab

## Context

Source describes an agent orchestration design (RUX) that treats the LLM as untrusted and introduces a deterministic Executor as a trust boundary enforcing schema/contract validation before tool execution to prevent hallucinated tool calls and silent failures.

## Decision

Add Executor Trust Boundary

## Consequences

This pattern/content has been added to the architecture lab repository.
