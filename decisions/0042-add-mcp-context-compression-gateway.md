# ADR-0042: Add MCP Context Compression Gateway

**Date**: 2026-04-27
**Status**: Accepted
**Pipeline**: pattern-lab

## Context

Source describes "Context Mode" as an MCP server interposed between an agent (Claude Code) and tool outputs to prevent context-window exhaustion by compressing tool responses (example claim: 315KB→5.4KB, 98% reduction).

## Decision

Add MCP Context Compression Gateway

## Consequences

This pattern/content has been added to the architecture lab repository.
