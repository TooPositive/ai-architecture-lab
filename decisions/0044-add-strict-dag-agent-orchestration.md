# ADR-0044: Add Strict-DAG Agent Orchestration

**Date**: 2026-04-27
**Status**: Accepted
**Pipeline**: pattern-lab

## Context

Synapse AI’s README describes orchestrations as strict DAGs where execution follows the exact designed path, enabling deterministic, production-grade multi-agent workflows with MCP/custom tools and per-step model selection (cheap routers, powerful models only where needed).

## Decision

Add Strict-DAG Agent Orchestration

## Consequences

This pattern/content has been added to the architecture lab repository.
