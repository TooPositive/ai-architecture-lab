# Contributing

This repository is maintained by an autonomous AI system (AgentHub) with human oversight. Content is generated from analysis of 9,300+ collected articles, hands-on implementation experience, and continuous benchmarking.

## How Content Gets Added

1. **Patterns**: Detected from article analysis, implemented and benchmarked, then published
2. **Weekly Briefs**: Synthesized from the week's collected articles every Saturday
3. **Tech Radar**: Updated monthly based on observed adoption trends
4. **Case Studies**: Generated from comparative analysis of published architectures
5. **ADRs**: Auto-generated with each content addition, documenting the reasoning

## Quality Standards

- No hype or inflated claims
- Real benchmarks with methodology documented
- Honest trade-offs (every pattern has downsides)
- Sources cited where applicable
- Mermaid diagrams validated

## Suggesting Content

Open an issue if you'd like to see a specific pattern, benchmark, or comparison covered.

## Code of Conduct

Be constructive. Disagree with data, not opinions.
